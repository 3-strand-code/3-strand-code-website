About Front End Web Development

In loose terms, front end web development encompasses “building the part of the web people see and use.” Front end development often includes elements of user interface design (UI) and user experience design (UX), though these are separate areas of expertise. Front end developers are usually responsible for taking UX and UI deliverables and creating real working interfaces.

The web used to be home to only websites. While the desktop hard drive was home to applications. In todays modern web, a web address may load a simple website or a full featured application without installing additional software. The user’s data and profile are all saved in “the cloud” and accessible on all their web enabled devices. A skilled front end web developer can build both websites and web applications.

Responsive design allows web app interfaces to adapt to mobile screen sizes. However, it should be noted that web apps are not native mobile apps. Native mobile apps are downloaded, typically, from an app store or market. Mobile app developers create native apps. Front end web developers make responsive web apps that run in a mobile device’s web browser.
